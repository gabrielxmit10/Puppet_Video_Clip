# Puppet - p5.js Music Video Animation

This is an animated music video created entirely with code using the **p5.js** library, synchronized to the song *"Puppet"* by John Michael Howell. 

This project was developed as the final assignment for my Computer Graphics college course at UFRJ. The assignment was open-ended—we had to code something of our choice using p5.js. I decided to explore the intersection of code and music, which led to the creation of this music video.

## 🚀 How to Run Locally

Due to copyright restrictions, the actual audio track is **not included** in this repository. To run the animation locally with the music, you will need to provide the song yourself.

1. **Clone the repository:**
   ```bash
   git clone https://github.com/gabrielxmit10/Puppet_Video_Clip.git
   cd Puppet_Video_Clip
   ```
2. **Add the Audio:**
   * Purchase or download the song *"Puppet"* by John Michael Howell.
   * Rename the downloaded file to exactly **`audio.mp3`**.
   * Place the file inside the `media/` folder.
3. **Run a Local Server:**
   * Because p5.js needs to load local media files (like textures and fonts), you cannot simply double-click `index.html`. You must serve it over a local web server to avoid browser CORS security errors.
   * *Recommended:* Use the **Live Server** extension in VSCode, or run `python -m http.server` in your terminal.
4. **Enjoy!**
   * Open the provided localhost link in your web browser.
